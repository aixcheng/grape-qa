# 葡萄病虫害问答系统 — 测试题集

## 说明
- **单跳问题**：查询单个实体或单个关系即可回答
- **多跳问题**：需要跨多个关系推理或结合多源信息
- **命中标准**：答案包含核心事实即可，不要求逐字匹配

---

## 一、单跳正向查询（实体 → 属性）

### Q1. 葡萄黑痘病有什么症状？
- **类型**：单跳正向
- **预期答案**：为害绿色幼嫩部分，果实出现"鸟眼"状病斑（中央灰白凹陷、边缘紫褐），叶片出现黑褐色斑点后穿孔，新梢出现长圆形凹陷病斑
- **检索路径**：Disease(葡萄黑痘病) → HAS_SYMPTOM

### Q2. 葡萄白腐病的病原是什么？
- **类型**：单跳正向
- **预期答案**：葡萄白腐壳孢菌（Coniella diplodiella），属半知菌亚门
- **检索路径**：Disease(葡萄白腐病) → HAS_PATHOGEN

### Q3. 葡萄根瘤蚜的形态特征是什么？
- **类型**：单跳正向
- **预期答案**：体小、椭圆形，有根瘤型和叶瘿型两种，成蚜赭褐色，幼蚜鲜红色
- **检索路径**：Pest(葡萄根瘤蚜) → HAS_CHARACTERISTIC

### Q4. 葡萄透翅蛾主要危害哪个部位？
- **类型**：单跳正向
- **预期答案**：枝蔓（幼虫蛀入嫩茎和枝蔓内部）
- **检索路径**：Pest(葡萄透翅蛾) → AFFECTS_PART

### Q5. 葡萄灰霉病的发病规律是什么？
- **类型**：单跳正向
- **预期答案**：低温高湿利于发病，主要为害花序和果实，花期和成熟期易感病
- **检索路径**：Disease(葡萄灰霉病) → HAS_RULE

### Q6. 葡萄霜霉病有哪些防治方法？
- **类型**：单跳正向
- **预期答案**：加强通风透光、清除病残体、药剂防治（波尔多液、甲霜灵等）
- **检索路径**：Disease(葡萄霜霉病) → HAS_METHOD

### Q7. 葡萄炭疽病的适宜环境条件是什么？
- **类型**：单跳正向
- **预期答案**：高温高湿，25-30℃最适宜，多雨季节易爆发
- **检索路径**：Disease(葡萄炭疽病) → FAVORED_BY

### Q8. 葡萄斑衣蜡蝉的发生时期是什么？
- **类型**：单跳正向
- **预期答案**：若虫期4-5月，成虫期6-9月
- **检索路径**：Pest(斑衣蜡蝉) → OCCURS_IN

### Q9. 葡萄白粉病的传播媒介是什么？
- **类型**：单跳正向
- **预期答案**：主要以分生孢子借风雨传播
- **检索路径**：Disease(葡萄白粉病) → TRANSMITTED_BY

### Q10. 葡萄酸腐病有什么防治药剂？
- **类型**：单跳正向
- **预期答案**：涉及杀菌剂和杀虫剂复配（具体药剂从 Treatment → USES_DRUG 路径获取）
- **检索路径**：Disease(葡萄酸腐病) → HAS_TREATMENT → USES_DRUG

### Q11. 葡萄黑痘病的别名是什么？
- **类型**：单跳正向
- **预期答案**：疮痂病、鸟眼病
- **检索路径**：Disease(葡萄黑痘病) → HAS_ALIAS

### Q12. 葡萄卷叶病由什么病原引起？
- **类型**：单跳正向
- **预期答案**：葡萄卷叶病毒（GLRaV），属长线病毒科
- **检索路径**：Disease(葡萄卷叶病) → HAS_PATHOGEN

## 二、单跳反向查询（属性 → 实体，诊断型）

### Q13. 叶片出现白色粉状霉层，可能是什么病？
- **类型**：单跳反向
- **预期答案**：葡萄白粉病
- **检索路径**：Symptom → SYMPTOM_OF → Disease

### Q14. 果实出现"鸟眼"状病斑是什么病？
- **类型**：单跳反向
- **预期答案**：葡萄黑痘病（又名鸟眼病）
- **检索路径**：Symptom → SYMPTOM_OF / Alias → ALIAS_OF

### Q15. "疮痂病"指的是哪种葡萄病害？
- **类型**：单跳反向
- **预期答案**：葡萄黑痘病
- **检索路径**：Alias(疮痂病) → ALIAS_OF → Disease

### Q16. 枝蔓内部被蛀空，蛀孔外有褐色虫粪，是什么虫害？
- **类型**：单跳反向
- **预期答案**：葡萄透翅蛾
- **检索路径**：Symptom → SYMPTOM_OF → Pest

### Q17. 葡萄根部形成根瘤、叶片黄化、树势衰弱，是什么害虫？
- **类型**：单跳反向
- **预期答案**：葡萄根瘤蚜
- **检索路径**：Symptom → SYMPTOM_OF → Pest

### Q18. 穗轴和果梗变黑褐色、组织变脆易断裂，是什么病？
- **类型**：单跳反向
- **预期答案**：葡萄白腐病
- **检索路径**：Part(穗轴) → PART_AFFECTED_BY + Symptom → SYMPTOM_OF

### Q19. 果皮出现铁锈斑，果粒纵向开裂，是什么虫害？
- **类型**：单跳反向
- **预期答案**：葡萄烟蓟马或葡萄短须螨（需根据更多特征区分）
- **检索路径**：Symptom → SYMPTOM_OF → Pest

### Q20. 叶片正面出现多角形黄色病斑，背面有白色霜状霉层，是什么病？
- **类型**：单跳反向
- **预期答案**：葡萄霜霉病
- **检索路径**：Symptom → SYMPTOM_OF → Disease

## 三、多跳推理问题

### Q21. 葡萄黑痘病在什么气候条件下最容易爆发？
- **类型**：多跳（2跳）
- **预期答案**：春夏多雨潮湿时发病最重，25-28℃高湿环境
- **检索路径**：Disease(葡萄黑痘病) → HAS_RULE + FAVORED_BY

### Q22. 防治葡萄霜霉病用什么药？具体怎么用？
- **类型**：多跳（2跳）
- **预期答案**：波尔多液、甲霜灵等，具体浓度和时期从 Treatment 节点获取
- **检索路径**：Disease(葡萄霜霉病) → HAS_TREATMENT → USES_DRUG

### Q23. 葡萄白腐病和葡萄黑痘病在果实症状上有什么区别？
- **类型**：多跳（多实体比较）
- **预期答案**：白腐病在果梗穗轴先发病、果粒软腐脱落；黑痘病为"鸟眼"状凹陷病斑、果粒变小酸涩
- **检索路径**：分别查询两种病害的症状再比较

### Q24. 最近连续下雨，湿度很高，我需要重点预防哪些葡萄病害？
- **类型**：多跳（条件反向多实体）
- **预期答案**：霜霉病、灰霉病、白腐病、炭疽病等高湿性病害
- **检索路径**：Condition(高湿) → 反向查询 → 多个 Disease

### Q25. 8月份果穗上出现褐色病斑、果粒易脱落，结合这个时期，可能是什么病？
- **类型**：多跳（时间+部位+症状）
- **预期答案**：葡萄白腐病（8月高温多雨，穗轴和果粒受害特征吻合）
- **检索路径**：Period(8月) + Part(果穗) + Symptom(褐色病斑/脱落)

### Q26. 葡萄透翅蛾造成的伤口可能诱发什么病害？
- **类型**：多跳（跨病虫害）
- **预期答案**：透翅蛾蛀食枝蔓造成伤口，可能诱发葡萄蔓割病等枝干病害
- **检索路径**：Pest(葡萄透翅蛾) → AGGRAVATES → Disease

### Q27. 防治葡萄根瘤蚜有哪些综合措施？
- **类型**：多跳（2跳）
- **预期答案**：农业防治（抗性砧木）+ 物理防治 + 化学防治（具体药剂）
- **检索路径**：Pest(葡萄根瘤蚜) → HAS_METHOD + HAS_TREATMENT → USES_DRUG

### Q28. 葡萄短须螨在什么季节危害最严重？为什么？
- **类型**：多跳（2跳）
- **预期答案**：7-8月为繁殖高峰期，最适温度29℃、湿度80%-85%
- **检索路径**：Pest(葡萄短须螨) → OCCURS_IN + FAVORED_BY

### Q29. 葡萄扇叶病的传播媒介是什么？这种媒介还会传播什么病害？
- **类型**：多跳（跨实体）
- **预期答案**：由剑线虫（Xiphinema）传播；还会传播葡萄黄脉病等多种病毒病
- **检索路径**：Disease(葡萄扇叶病) → TRANSMITTED_BY → Vector

### Q30. 如果葡萄园同时有葡萄透翅蛾和葡萄蔓割病发生，防治策略应该如何协调？
- **类型**：多跳（综合推理）
- **预期答案**：先防虫（阻断伤口来源）再防病（保护枝干），透翅蛾防治+蔓割病药剂保护结合
- **检索路径**：分别获取虫害防治方法和病害防治方法后综合

## 四、文本检索依赖型（需要向量检索补充）

### Q31. 葡萄黑痘病的苗木消毒具体用什么药剂？
- **类型**：混合（图谱+文本）
- **预期答案**：10%-15%硫酸铵溶液、3%-5%硫酸铜溶液、3-5波美度石硫合剂等
- **检索路径**：图查询 Treatment + 文本检索补充细节

### Q32. 葡萄白粉病防治中，石硫合剂应该在什么时期使用？
- **类型**：混合（图谱+文本）
- **预期答案**：葡萄发芽前喷3-5波美度石硫合剂铲除越冬病菌
- **检索路径**：图查询 + 文本检索

### Q33. 巨峰葡萄对哪些病害有较好的抗性？
- **类型**：混合（需要从文本中提取特定品种信息）
- **预期答案**：对黑痘病属中抗类型，对白粉病较抗病
- **检索路径**：向量检索文本中"巨峰"相关内容

### Q34. 葡萄灰霉病在果实上的症状和穗轴褐枯病有什么区别？
- **类型**：混合（多实体症状比较）
- **预期答案**：灰霉病花序和幼果腐烂、生灰色霉层；穗轴褐枯病穗轴变褐坏死、不产生霉层
- **检索路径**：分查两个实体的症状描述

### Q35. 用50%多菌灵防治葡萄黑痘病，应该用多少倍液？
- **类型**：混合（精确数值）
- **预期答案**：50%多菌灵可湿性粉剂600-800倍液
- **检索路径**：Disease → Treatment（含浓度剂量字段）

## 五、边角测试

### Q36. 苹果黑星病如何防治？（系统外问题）
- **类型**：边界测试
- **预期行为**：提示"根据现有知识暂时无法回答"或"这是关于苹果的问题，我主要回答葡萄相关"

### Q37. 葡萄黑痘病和葡萄黑腐病是同一种病吗？
- **类型**：对比推理
- **预期答案**：不是，两者病原不同、症状不同、发病规律不同

### Q38. 葡萄有哪几种常见的果实病害？
- **类型**：聚合查询
- **预期答案**：黑痘病、白腐病、炭疽病、灰霉病、酸腐病、黑腐病等

### Q39. 葡萄二星叶蝉和葡萄斑衣蜡蝉有什么区别？
- **类型**：多实体比较
- **预期答案**：两种不同害虫，形态特征、危害方式、发生时期不同

### Q40. "白腐病"这个名称的由来说明它的什么发病特征？
- **类型**：语义推理
- **预期答案**：因受害果实呈白色腐烂状而得名
