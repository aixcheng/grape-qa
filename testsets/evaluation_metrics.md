# 葡萄病虫害知识图谱问答系统测试指标

本文档用于毕业设计测试章节，可与 `grape_qa_testset.csv` 和 `run_grape_qa_tests.ps1` 配套使用。

## 1. 功能性指标

| 指标 | 含义 | 建议计算方式 |
| --- | --- | --- |
| 功能测试通过率 | 登录、知识列表、知识详情、问答、历史记录、图谱展示等功能是否正常 | 通过用例数 / 总功能用例数 |
| 知识详情完整率 | 知识百科接口是否能返回指定病虫害的关键章节 | 完整返回章节的实体数 / 测试实体数 |
| 图谱可视化可用性 | 节点、边、搜索、缩放、点击详情是否可用 | 人工检查或前端自动化检查通过率 |

## 2. 知识图谱质量指标

| 指标 | 含义 | 建议计算方式 |
| --- | --- | --- |
| 实体类别覆盖率 | 是否覆盖 Disease、Pest、Alias、Symptom、Part、Rule、Method、Drug、Treatment、Pathogen、Characteristic、Condition、Period、Vector 等 14 类实体 | 已出现实体类别数 / 14 |
| 关系类别覆盖率 | 是否覆盖 HAS_SYMPTOM、HAS_METHOD、HAS_PATHOGEN、USES_DRUG 等 16 类关系 | 已出现关系类别数 / 16 |
| 实体覆盖率 | `data/texts` 中病虫害是否都进入知识库或图谱 | 已入库实体数 / 文本实体数 |
| 关系准确率 | 抽样检查图谱边是否符合原文 | 正确关系数 / 抽样关系数 |
| 孤立节点率 | 是否存在没有任何关系的节点 | 孤立节点数 / 总节点数，越低越好 |

## 3. 检索与问答指标

| 指标 | 含义 | 建议计算方式 |
| --- | --- | --- |
| 问答关键词命中率 | 回答中是否包含测试集设定的关键事实 | 命中关键词数 / 期望关键词数 |
| 问答用例通过率 | 回答命中不少于 `min_keyword_hits` 个关键词即通过 | 通过问答数 / 总问答数 |
| Top-k 召回率 | 向量或图谱检索前 k 条是否包含目标实体/章节 | 命中目标的查询数 / 总查询数 |
| 反向诊断准确率 | 根据症状反推病虫害是否正确 | 正确诊断数 / 反向诊断用例数 |
| 药剂剂量准确率 | 涉及药剂、倍液、时期、用量的问题是否回答正确 | 正确剂量用例数 / 剂量用例数 |
| 未知问题拒答率 | 知识库不存在的问题是否避免胡编 | 正确拒答数 / 未知问题数 |

## 4. 性能指标

| 指标 | 含义 | 建议目标 |
| --- | --- | --- |
| 知识列表接口响应时间 | `/api/knowledge/list` 平均耗时 | 小于 500 ms |
| 知识详情接口响应时间 | `/api/knowledge/<name>` 平均耗时 | 小于 500 ms |
| 问答接口响应时间 | `/api/chat` 平均耗时 | 本地/外部大模型场景建议小于 10 s |
| 前端首屏加载时间 | 首页或图谱页可交互时间 | 小于 3 s |

## 5. 稳定性与安全指标

| 指标 | 含义 |
| --- | --- |
| 异常处理通过率 | Neo4j、ChromaDB、大模型 API 不可用时是否返回友好错误 |
| 连续问答稳定性 | 连续多轮提问是否正常响应、不崩溃 |
| 权限控制正确率 | 未登录用户是否不能访问需要权限的历史记录接口 |
| 输入安全性 | 搜索框、问答框是否能处理特殊字符，避免 XSS 或后端异常 |

## 6. 当前测试集覆盖说明

`grape_qa_testset.csv` 当前包含 98 条问答用例，覆盖以下类型：

- `symptom`：病害/虫害症状查询
- `characteristic`：害虫形态特征查询
- `method`：防治方法查询
- `reverse_diagnosis`：由症状反推病虫害
- `alias`：别名识别
- `treatment_dosage`：药剂和倍液测试
- `condition_reasoning`：环境条件推理
- `pathogen`：病原查询
- `transmission` / `vector_reasoning`：传播方式和媒介查询
- `unknown_guardrail`：未知问题拒答测试

推荐毕业设计报告中重点展示：功能测试通过率、知识详情完整率、问答用例通过率、反向诊断准确率、药剂剂量准确率、平均响应时间。
